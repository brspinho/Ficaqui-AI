# 🏙️ Projeto Ficaqui: Ecossistema de Inteligência Urbana

O Ficaqui é uma plataforma de decisão baseada em dados que conecta o poder público (Prefeitura), o capital privado (Fundos de Investimento) e o cidadão para reocupar o Centro de Aracaju. Este MVP é focado no Pilar de **Cérebro (Ficaqui AI)**, cruzando microdados de transporte, demografia e segurança para instruir ações de **Políticas Públicas** e atração do **Fundo Imobiliário (Músculo)** focando em Moradia de Uso Misto.

## 🚀 Como Rodar o Programa (Dashboard B2G)

Siga os passos abaixo para executar e visualizar o dashboard do Streamlit na sua máquina:

### 1. Pré-requisitos
Certifique-se de ter o [Python](https://www.python.org/downloads/) instalado na sua máquina.

### 2. Instalação das Dependências
Abra o terminal (PowerShell ou Prompt de Comando) na pasta do projeto (`Ficaqui-AI`) e instale as bibliotecas essenciais rodando o seguinte comando:

```bash
pip install streamlit pandas folium streamlit-folium plotly
```

### 3. Executando o Dashboard
Com as dependências instaladas, inicie o servidor do Streamlit com o comando:

```bash
python -m streamlit run main.py
```
> **Nota:** Se for a primeira vez que você roda o Streamlit na sua máquina, ele pode perguntar se você deseja cadastrar um email no terminal. Você pode apenas deixar o campo em branco e pressionar **Enter** para pular e seguir normalmente.

### 4. Acessando a Plataforma
Após o carregamento, a aplicação geralmente abrirá o seu navegador padrão automaticamente. Caso contrário, acesse através do link abaixo:
👉 **[http://localhost:8501](http://localhost:8501)**
